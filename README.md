# 🌬️ Мониторинг качества воздуха на ESP32 + FreeRTOS

Проект домашней метеостанции на базе **ESP32** с операционной системой реального времени **FreeRTOS**. Устройство считывает данные с датчиков **AGS02MA** (TVOC) и **MQ-135** (CO₂, пары бензина, аммиак) и отображает их через веб-интерфейс по Wi-Fi.

## Возможности

- Опрос датчиков AGS02MA (I²C) и MQ-135 (АЦП) каждые 5 секунд
- Параллельная работа трёх задач FreeRTOS: опрос датчиков, веб-сервер, счётчик времени
- Синхронизация задач через Event Groups и очереди
- Красивый адаптивный веб-интерфейс с автообновлением (Refresh: 5 сек)
- Оценка качества воздуха (Отличный / Средний / Плохой / Очень плохой)
- Переподключение к Wi-Fi при обрыве соединения

## Используемый стек

- **Микроконтроллер:** ESP32-D0WDQ6 (плата NodeMCU-32S)
- **ОСРВ:** FreeRTOS (задачи, очереди, Event Groups, семафоры, таймеры)
- **Фреймворк:** ESP-IDF v6.0.1
- **Язык:** C
- **Протоколы:** I²C, ADC, TCP/IP (lwIP), HTTP
- **Инструменты:** VS Code + ESP-IDF Extension, CMake, Git

## Аппаратная часть

| Датчик   | Параметр          | Подключение          |
|----------|-------------------|----------------------|
| AGS02MA  | TVOC (ppb, µg/m³) | I²C: SDA=GPIO21, SCL=GPIO22 |
| MQ-135   | Напряжение (АЦП)  | AO → GPIO36 (ADC1_CH0)       |

## Как собрать и запустить

1. Установить ESP-IDF v6.0.1 и VS Code с расширением Espressif IDF.
2. Клонировать репозиторий.
3. В файле `main.c` заменить `WIFI_SSID` и `WIFI_PASS` на свои.
4. Собрать и прошить: `idf.py build flash monitor`.
5. Открыть в браузере `http://IP_адрес_платы`.
